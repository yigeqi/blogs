# 列表（列表会自动有缩进）

```

```

```
1. 列表1
2. 列表2
3. 列表3
```

2. 列表2
3. 列表3

#### 嵌套列表
上一级列表内容后面加三个空格后再换行即可。
1. 列表1   
1.1  asfadf
1.2 sdf s
2. 列表2   
2.1 sss
3. 列表3

# 表格
```
表头1|表头2|表头3
-|:-:|-:
c|c|c
content|content|content|
```   
注意用于分割表头和内容的第二行：-的个数不限，   
文字默认居左，   
-两边加：表示文字居中   
-右边加：表示文字居右
